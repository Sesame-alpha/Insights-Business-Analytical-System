# Insights Business Analytical System

**Business Intelligence Decision Support Tool**

A powerful, client‑side business analytics dashboard that helps you track sales, calculate profits, visualize trends, and generate actionable insights—all without any server, database, or account setup.

Key Features

- **Multi‑Currency Support** – Automatically detects currency symbols in imported data (supports all world currencies) or lets you set a custom symbol.
- **Transaction Management** – Add, edit, or delete records with date, product, sales, cost. Profit auto‑calculated.
- **Real‑time KPIs** – Total Sales, Total Profit, Average Margin, and dynamic Sales Trend (Strong Upward, Slight Upward, Stable, etc.).
- **Interactive Charts** – Bar (sales) + line (profit) chart using Chart.js. Grouped by date.
- **AI Insights** – Highlights profitability, top performer, and margin health.
- **Executive Summary** – Shows key metrics, top 3 products, bottom 3 products, average profit per transaction.
- **Date Range Filter** – Focus on any time period. All charts and KPIs update instantly.
- **CSV & Excel Import/Export** – Upload files with flexible column detection (no need to match exact headers). No file size limit (browser‑dependent).
- **LocalStorage Persistence** – All data saved automatically in your browser. Close and reopen—nothing lost.
- **Fully Responsive** – Works on desktops, tablets, and smartphones.
- **No Backend, No Installation** – Just open the HTML file and start using.


Tech Stack
- **HTML5** – Structure and mark-up 

- **CSS3** – External custom styling (clean corporate look)
- **Bootstrap 5** – Responsive grid and components
- **JavaScript (ES6)** – Core logic, trend detection (linear regression), CSV parsing, LocalStorage
- **Chart.js** – Interactive sales & profit visualization
- **Font Awesome** – Icon set for better CSV
